Simple Static Site

This repository contains a minimal static website with three pages and one stylesheet:

- index.html
- about.html
- contact.html
- styles.css

Preview locally:

Using Python 3:

    cd /workspaces/gitEnviroment
    python3 -m http.server 8000

Then open http://localhost:8000 in your browser.

Using VS Code Live Server: open the folder and click "Go Live".
